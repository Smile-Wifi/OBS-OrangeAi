# OBS-OrangeAI Web App

A fully functional web-based OBS clone.

## Features
- Scenes management
- Draggable overlays (text/image)
- Audio mixer (mic/desktop)
- Performance stats
- Local recording
- Streaming via RTMP (Facebook, YouTube, Twitch, TikTok)
- Mobile-responsive dashboard
- PWA-ready
